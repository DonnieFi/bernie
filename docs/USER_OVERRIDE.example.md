# USER_OVERRIDE (example)

Copy to `docs/USER_OVERRIDE.md` (or set `hot_memory.user_override_path`).

**Human-edited only.** Bernie tools will not write this file. Facts here always
win over agent-appended `context.md` / person docs.

## Examples

- Never schedule school events after 9pm without parent OK
- Child1 is allergic to tree nuts (confirm with parents before meals)
